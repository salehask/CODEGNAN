assignment_indLocal Variables

Local variables are declared inside methods, constructors, or blocks. They are created when the method or block is executed and destroyed once it finishes. They are only accessible within the block or method where defined.

public void exampleMethod() {
    int localVar = 10;  // Local Variable
    System.out.println(localVar);
}
Method Scope

localVar exists only here inside this method.

account_treeGlobal (Instance) Variables

Global variables (also called instance variables) are declared inside a class but outside any method or constructor. Each instance of the class has its own copy. They are accessible by all instance methods of the class.

class Example {
    int globalVar;  // Instance Variable

    public void printVar() {
        System.out.println(globalVar);
    }
}
Class Scope (Instance)

globalVar exists as a member of each object instance.

Accessible by all instance methods during the lifetime of the object.

storageStatic Variables

Static variables are declared with the static keyword inside a class but outside any method. They belong to the class itself rather than to any instance. All instances share the same static variable.

class Example {
    static int staticVar;  // Static Variable

    public static void printStaticVar() {
        System.out.println(staticVar);
    }
}
Class Scope (Static)

staticVar exists at class level, shared by all instances.

Accessible without creating an object instance.
